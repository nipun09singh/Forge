# SaaS Support Pro

AI-powered customer support, onboarding, and retention agency for SaaS products

**Domain:** SaaS customer support with technical troubleshooting, billing, and onboarding

Generated by [Forge](https://github.com/forge) — AI Agency Meta-Factory.

## Teams

### Support Team
Front-line customer support handling tickets and inquiries

**Lead:** Support Lead (Customer Support Team Lead)

**Agents:**
- **Technical Support** — Technical Support Engineer (AgentRole.SPECIALIST)
- **Billing Support** — Billing & Account Specialist (AgentRole.SPECIALIST)
- **Onboarding Guide** — Customer Onboarding Specialist (AgentRole.SUPPORT)
### Quality & Improvement
Cross-cutting team responsible for quality assurance, performance monitoring, continuous improvement, and request intake. These agents operate across all other teams to ensure agency-wide quality standards.

**Lead:** Intake Coordinator (Request Intake & Routing Coordinator)

**Agents:**
- **Strategic Planner** — Strategic Task Planner & Orchestrator (AgentRole.COORDINATOR)
- **QA Reviewer** — Quality Assurance Reviewer (AgentRole.REVIEWER)
- **Self-Improvement Agent** — Performance & Improvement Analyst (AgentRole.ANALYST)
- **Analytics Agent** — Agency Analytics & Reporting Specialist (AgentRole.ANALYST)
### Revenue & Growth
Revenue-focused team responsible for customer acquisition, retention, monetization optimization, and growth hacking. These agents ensure the agency drives maximum business value and revenue generation.

**Lead:** Revenue Optimizer (Revenue Optimization & Monetization Strategist)

**Agents:**
- **Growth Hacker** — Growth & Viral Acquisition Specialist (AgentRole.SPECIALIST)
- **Customer Success Agent** — Customer Success & Retention Specialist (AgentRole.SUPPORT)
- **Lead Generation Agent** — Lead Generation & Pipeline Builder (AgentRole.SPECIALIST)

## Quick Start

```bash
# Install dependencies
pip install -r requirements.txt

# Set your API key
export OPENAI_API_KEY="your-key"

# Run interactively
python main.py

# Or run as API server
uvicorn api_server:app --host 0.0.0.0 --port 8000
```

## API Endpoints

- `POST /api/task` — Execute a task
- `GET /api/status` — Get agency status
- `GET /health` — Health check
- `POST /api/task` — Submit a support task
- `POST /api/tickets` — Create a support ticket
- `POST /api/onboard` — Start customer onboarding

## Docker

```bash
docker compose up --build
```
