# CodeFusionAI

CodeFusionAI is an AI-powered software development agency that leverages machine learning to write code, run tests, and deploy applications far more efficiently than human teams, offering game-changing cost savings and speed for businesses of all sizes.

**Domain:** AI software development agency that writes code, runs tests, and deploys applications

## Quick Start

```bash
# Install dependencies
pip install -r requirements.txt

# Set your API key
export OPENAI_API_KEY="your-key"

# Run interactively
python main.py

# Or run as API server
uvicorn api_server:app --host 0.0.0.0 --port 8000
```

## API Endpoints

- `POST /api/task` — Execute a task
- `GET /api/status` — Get agency status
- `GET /health` — Health check
- `POST /api/task` — For general tasks
- `POST /api/code` — For automated code writing
- `POST /api/test` — For automated testing
- `POST /api/deploy` — For automated deployment
- `POST /api/bug-detection` — For automated bug detection and resolution
- `POST /api/optimize` — For code optimization
- `POST /api/customer` — For customer acquisition and retention
- `POST /api/selling` — For upselling and cross-selling services
- `POST /api/insights` — For data-driven insights and recommendations
