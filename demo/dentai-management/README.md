## Quick Start

```bash
# Install dependencies
pip install -r requirements.txt

# Set your API key
export OPENAI_API_KEY="your-key"

# Run interactively
python main.py

# Or run as API server
uvicorn api_server:app --host 0.0.0.0 --port 8000
```

## API Endpoints

- `POST /api/task` — Execute a task
- `GET /api/status` — Get agency status
- `GET /health` — Health check
- `POST /api/appointment/schedule` — Automated appointment scheduling and reminders
- `POST /api/patient/triage` — AI-powered patient triaging and symptom assessment
- `POST /api/supply/predictive-analytics` — Predictive analytics for supply chain management
- `POST /api/treatment/plan` — AI-assisted treatment planning and cost estimation
- `POST /api/revenue/optimize` — Revenue optimization through cross-selling and upselling
- `POST /api/customer/retention` — Customer retention strategies
- `POST /api/billing/process` — Automated billing and insurance claims processing
- `POST /api/task` — General tasks
